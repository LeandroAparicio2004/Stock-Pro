> Sistema de control de stock profesional, portable y multi-empresa. Desarrollado en Python con interfaz gráfica.

---

## Características

- 🏢 **Multi-empresa** — cada cliente tiene su propio archivo `.spdb`, sin datos compartidos
- 📦 **ABM de productos** — alta, baja y modificación con categorías, precios y stock mínimo
- 🔄 **Movimientos** — entradas, salidas y ajustes con historial completo
- 🔔 **Alertas** — panel de productos bajo stock mínimo
- 📊 **Estadísticas** — gráficos de movimientos, productos más activos y distribución por categoría
- 🔍 **Filtros** — por nombre, código, categoría y estado
- 📤 **Exportar** — inventario y movimientos en Excel y PDF
- 📥 **Importar** — carga masiva de productos desde cualquier Excel, sin formato fijo
- 🖱️ **Doble clic** — registrá movimientos directamente desde la tabla

## Instalación y uso

### Ejecutable (usuarios finales)
Descargá el `.exe` desde [Releases](../../releases) y ejecutalo directamente. No requiere Python ni ninguna instalación adicional.

# Dependencias
pip install customtkinter pillow matplotlib openpyxl reportlab

# Ejecutar
python app.py
